Spend Distributor

Languages Used : Java with spring boot framework, javascript, jquery, html, css for frontend.

Back-end of spend distributor application is developed with java with spring boot in spring MVC pattern. I have used spring security with JWT token for the authentication. MySql database is used for storing and organizing the data. Front-end is developed with the Angular js.

more details

Spend Distributor Application is designed to distribute the room, trip kind of expenditures among the group members. It facilates the CRUD operations on users, groups and spends.

Admin can create a group of respective members for spend distribution. All authenticated members who spend money, can mark their spend's entries in the respective group. In the analysis section the distributed spends can be analysed.

The graphical representation is provided by integrating fussion charts for better understanding.
